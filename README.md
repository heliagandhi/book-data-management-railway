# book-data-management-railway
Proyek untuk mengelola input data buku beserta kategorinya menggunakan relational database PostgreSQL.  Aplikasi ini dirancang untuk mendukung operasi CRUD pada data buku dan kategori.  Project di-deploy ke production menggunakan Railway.  Tujuan utama: membangun sistem manajemen data buku yang sederhana, efisien, dan siap digunakan di production.
